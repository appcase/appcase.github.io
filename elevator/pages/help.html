Elevator Controller : Help
  
As the elevator controller of a ten storeyed building with seven elevators, your task is to assign the best elevator option to each of the visitors to minimize their waiting time in the queue.

The game dashboard shows ten floors F:0 to F:10 with the number of visitors waiting, in the red box beside each floor number. It also displays the seven elevators E:1 to E:7 in the panel below. The grid displays the movement of the elevators in-between the floors.

Tap on a floor number and it will list all the visitors waiting there along with their id, current waiting time, the floor and direction where they intend to go. Assign each of them the best elevator available. To help you decide, the elevator panel displays the current position of each of the elevators with respect to the floor the vistor is currently waiting. Tap on an elevator to view the passenger details and infer information like the floors where the elevator has to make a stop.

Each visitor has a hidden random expected waiting time. At the end of the visitor movement the game tracks the actual waiting time and compares that with the expected. This is the 'Op Ratio'.

Try to keep the 'Op Ratio' below or equal to 1. It signifies that all the visitors were moved within their expected waiting time.

If the visitor crowd at any of the floors exceed 50, the game ends. Otherwise, this is a simple game that requires a little bit of memory, reflexes and mental mathematics. Enjoy playing...

P.S: This game does not send any data to any server but saves locally in your browser. You can close the game any time and resume it by reopening.
