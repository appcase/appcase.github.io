Elevator Controller : Help
  
As the elevator controller of a multi storeyed building, you have to assign the best elevator option to your moving visitors to minimize their waiting time in the queue.

You game dashboard shows each of the floors along with the number of visitors waiting there. It also displays the elevator movements in-between the floors.

Tap on a floor and it will list all the visitors there. Assign an elevator to each of them by deciding which one will be the best for the visitor.

Each visitor has a hidden random expected waiting time. At the end of the visitor movement the game tracks the actual waiting time and compares that with the expected. This is the 'Op Ratio'.

Try to keep the 'Op Ratio' below or equal to 1. It signifies that all the visitors were moved within their expected waiting time.

If the visitor crowd at any of the floors exceed 100, the game ends. Otherwise, this is a simple game that requires eye and finger coordination and a little bit of mental mathematics. Enjoy playing...

P.S: This game does not send any data to any server but stores locally in your browser. You can close the game any time. You can resumes the game by reopening it.
